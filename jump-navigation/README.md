# Jump Navigation

Quickly jump between messages in TypingMind using keyboard shortcuts.

## Keyboard Shortcuts

### Mode 1: Jump All Messages (User & AI)
| Platform | Shortcut |
|----------|----------|
| Windows/Linux | `Ctrl + Up/Down Arrow` |
| Mac | `Cmd + Up/Down Arrow` |

### Mode 2: Jump User Prompts Only
| Platform | Shortcut |
|----------|----------|
| Windows/Linux | `Ctrl + Shift + Up/Down Arrow` |
| Mac | `Cmd + Shift + Up/Down Arrow` |

## Features

- Smooth scrolling between messages
- Two navigation modes for flexibility
- Works seamlessly with TypingMind's interface
