# TypingMind Extensions

A collection of custom extensions for [TypingMind](https://www.typingmind.com/).

## Available Extensions

| Extension | Description | Install Link |
|-----------|-------------|--------------|
| [Jump Navigation](./jump-navigation/) | Jump between messages using keyboard shortcuts | [Install](#jump-navigation) |

---

## Installation

1. Open TypingMind
2. Go to **Settings** → **Extensions**
3. Click **Add Extension**
4. Paste the extension URL (see below)
5. Save and refresh TypingMind

---

## Jump Navigation

Quickly navigate between messages using keyboard shortcuts.

**Extension URL:**

https://cdn.jsdelivr.net/gh/NocturnalKernel/typingmind-extensions@latest/jump-navigation/jump-navigation.js


**Shortcuts:**
- `Ctrl/Cmd + Up/Down` — Jump between all messages
- `Ctrl/Cmd + Shift + Up/Down` — Jump between user prompts only

[Full documentation](./jump-navigation/)
